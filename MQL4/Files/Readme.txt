Советник работает полностью автоматически: торгует, делает оптимизацию периодически, сам себя тестирует. Советник контролирует спред, если спред повышенный больше двух раз тогда торговля не идёт.

Инструкции по использованию:

0. Выключаем мт4. 

1. Очищаем папку MQL4/Files на своём компьютере. Пункт 1 только для уже существующих пользователей, для новых пользователей этот пункт делать не обязательно.

2. Копируем папку MQL4 из релиза в папку терминала МТ4. Включаем мт4.

3. Настраиваем окно "Обзор рынка / MarketWatch" - оставляем нужные валюты, остальные удаляем.

4. В меню мт4 "Сервис" / "Настройки" / "Графики" / "Макс.баров в истории" и "Макс.баров в окне" устанавливаем "1600" и перезагружаем мт4.

5. В меню мт4 "Сервис" / "Архив котировок" выбираем для каждого символа из "Обзор рынка / MarketWatch" таймфремы 15,30,60 и загружаем для каждого символа историю котировок отдельно.Загружать историю следует только для таймфреймов 15,30,60 для остальных история не требуется. 

6. Перетаскиваем советник MarketTrader.mq4 на график M5 любой валютной пары, указывая в настройках ключ. Ждём 2 минуты, если появилась красная надпись "Исправьте ошибки!" - докачиваем историю по валютам которые серые на графике или удаляем эти валюты из файла contests.txt. Оптимизатор закрывает мт4 для сохранения котировок на диск и производит расчёты, после этого снова сам открывает мт4 и продолжает торговать на новых расчётах. Оптимизатор срабатывает раз в 24 часа, на графике идёт обратный отсчёт.

7. Сохраняем профиль в мт4 под именем Default.
