Советник работает полностью автоматически: торгует, делает оптимизацию периодически. Советник контролирует спред, если спред повышенный тогда торговля не идёт.

Инструкции по использованию:

0. Очищаем папку MQL4/Files на своём компьютере. Пункт 0 только для уже существующих пользователей, для новых пользователей этот пункт делать не обязательно.

1. Копируем папку MQL4 из релиза в папку терминала МТ4.

2. Настраиваем окно "Обзор рынка / MarketWatch" - оставляем нужные валюты, остальные удаляем.

3. В меню мт4 "Сервис" / "Настройки" / "Графики" / "Макс.баров в истории" и "Макс.баров в окне" устанавливаем "1600" и перезагружаем мт4.

4. В меню мт4 "Сервис" / "Архив котировок" выбираем для каждого символа из "Обзор рынка / MarketWatch" таймфремы 15,30,60,240,1440 и загружаем для каждого символа историю котировок отдельно.Загружать историю следует только для таймфреймов 5,15,30,60, для остальных история не требуется. Если всё сделано верно, то для 28 валютных пар место на диске под историю займёт 1432 мегабайт для 6-ти таймфреймов.

5. Перетаскиваем советник MarketTrader.mq4 на график. Ждём 2 минуты, если появилась красная надпись "Исправьте ошибки!" - докачиваем историю по валютам которые серые на графике или удаляем эти валюты из файла context.txt. Оптимизатор закрывает мт4 для сохранения котировок на диск и производит расчёты, после этого снова сам открывает мт4 и продолжает торговать на новых расчётах. Оптимизатор срабатывает раз в 24 часа, на графике идёт обратный отсчёт.

6. Сохраняем профиль в мт4 под именем Default.
