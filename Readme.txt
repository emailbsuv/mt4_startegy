
Инструкции по использованию:

0. Смотрим видеоинструкции по ручной оптимизации MarketTrader в ютубе https://youtu.be/w_bDxYcnqTY

1. Копируем папки MQL4 и tester в папку мт4.

2. Проводим ручную оптимизацию советником MarketTrader-v1.084-opt.ex4 в оптимизаторе мт4 (MarketTrader-v1.085-opt.ex5 в оптимизаторе мт5). Настройки оптимизатора mt4 загружаем из папки tester файл Optimization.set . Модель оптимизации: по ценам открытия баров, период оптимизации: H1, спред: текущий, диапазон дат оптимизации: 1 год. Результаты оптимизации сохраняем в папку MQL4/Files с именем файлов на примере usdchf_h1.set для каждого прооптимизированного символа. Оптимизацию достаточно делать 1 раз в неделю.

3. Перетаскиваем советник MarketTrader-v1.09.ex4 на любой график любой валютной пары, указывая в настройках ключ. Разрешаем советнику торговать и использовать DLL в мт4, mt5.

4. Сохраняем профиль в мт4,mt5 под именем Default.
