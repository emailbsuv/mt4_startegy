Инструкции по установке, использованию и усовершенствованию.

Установка.
1. Копируем папку MQL5 в папку терминала.
2. Устанавливаем python: https://www.python.org/downloads/  . При установке python устанавливаем галочку "Проставить пути в окружении"
3. Устанавливаем pip: сохраняем файл https://bootstrap.pypa.io/get-pip.py , открываем cmd и вводим python get-pip.py
4. Устанавливаем tensorflow: открываем cmd и вводим pip install tensorflow
5. Устанавливаем numpy: открываем cmd и вводим pip install numpy

Использование.
1. Удаляем валюты из окна "Обзор рынка" которых нет в списке:
AUDCAD
AUDCHF
AUDNZD
AUDUSD

GBPUSD

EURNZD

GBPCHF
NZDCAD
NZDCHF
NZDUSD
USDCAD
AUDJPY
CADJPY
CHFJPY
EURJPY
GBPJPY
NZDJPY
USDJPY
2. Перетаскиваем MarketTrader.mq5 на график любой пары с таймфреймом М15
3. Ждём 2-5 минут пока загрузится 1400 баров D1 каждого символа из окна "Обзор рынка", зеленые пары - история подгружена, серые - нехватает 1400 баров в истории - либо вручную подгрузить либо удалить из окна "Обзор рынка".
4. В 23:00 каждый день по серверному времени мт5, кроме выходных, запускается TensorFlow код обучения модели с последующими предсказаниями. Предсказываются всего 2 сделки - одна BUY, вторая SELL. Результат ложится в файл predict.txt с последующим использованием в MarketTrader.mq5 . Расширенный результат добавляется в predict.log для последующих разработок и исследований.
5. В 00:20 каждый день по серверному времени мт5, кроме выходных, на основе расчетов TensorFlow открываются две сделки одна BUY и вторая SELL с тейкпрофитом 250 пунктов и стоплосом 1250 пунктов (не отложники).
6. Сохраняем профиль мт5 под именем Default

Усовершенствование.
1. Для усовершенствования кода предсказания цен можно использовать два файла MarketTrader.mq5 и ohlc-LSTM_D1.py (основной)
2. Для усовершенствования кода предсказания цен в ohlc-LSTM_D1.py можно экспериментировать с количеством слоёв, нейронов, типом моделей, эпох, количеством входных данных для обучения, типом входных данных для обучения модели - можно попробовать значения индикаторов мт5 в качестве входных данных для обучения модели.
3. Для подстройки торговых команд мт5 под результаты расчётов TensorFlow можно использовать усовершенствование кода MarketTrader.mq5 .
